Конструкция if-else и тернарный оператор

Оператор if

Оператор if позволяет выполнить блок кода, только если определенное условие истинно. Синтаксис оператора if выглядит следующим образом:

1

if (условие) { //блок кода, который выполнится, если условие истинно }

Давайте рассмотрим пример, в котором используется оператор if. Допустим, мы хотим проверить, является ли число положительным. Если это так, то мы выводим сообщение на экран. В противном случае, мы не выполняем никаких действий.

1

int number = 5; if (number > 0) { Console.WriteLine("Число положительное."); }

В данном примере мы определяем переменную number со значением 5. Затем мы проверяем, является ли число больше нуля. Если это так, то мы выводим сообщение "Число положительное." на экран.

Оператор if/else

Оператор if/else позволяет выполнить один блок кода, если условие истинно, и другой блок кода, если условие ложно. Синтаксис оператора if/else выглядит следующим образом:

1

if (условие) { //блок кода, который выполнится, если условие истинно } else { //блок кода, который выполнится, если условие ложно }

Давайте рассмотрим пример, в котором используется оператор if/else. Допустим, мы хотим проверить, является ли число положительным или отрицательным. Если число положительное, то мы выводим сообщение "Число положительное.", а если число отрицательное, то мы выводим сообщение "Число отрицательное."

1

int number = -3; if (number > 0) { Console.WriteLine("Число положительное."); } else { Console.WriteLine("Число отрицательное."); }

В данном примере мы определяем переменную number со значением -3. Затем мы проверяем, является ли число больше нуля. Если это так, то мы выводим сообщение "Число положительное." на экран. В противном случае, мы выводим сообщение "Число отрицательное.".

Оператор if/else if/else

Оператор if/else if/else позволяет проверять несколько условий и выполнять соответвующий блок кода в зависимости от выполнения каждого условия. Синтаксис оператора if/else if/else выглядит следующим образом:

1

if (условие1) { //блок кода, который выполнится, если условие1 истинно } else if (условие2) { //блок кода, который выполнится, если условие1 ложно, а условие2 истинно } else { //блок кода, который выполнится, если ни одно из условий не истинно }

Давайте рассмотрим пример, в котором используется оператор if/else if/else. Допустим, мы хотим определить, какую стоимость доставки следует применить в зависимости от выбранного способа доставки. Если выбран способ доставки "авиа", то стоимость доставки составит 1000 рублей. Если выбран способ доставки "ж/д", то стоимость доставки составит 500 рублей. Если выбран способ доставки "автобус", то стоимость доставки составит 300 рублей. В противном случае, мы выводим сообщение "Выберите способ доставки."

1

string deliveryMethod = "авиа"; int deliveryCost = 0; if (deliveryMethod == "авиа") { deliveryCost = 1000; } else if (deliveryMethod == "ж/д") { deliveryCost = 500; } else if (deliveryMethod == "автобус") { deliveryCost = 300; } else { Console.WriteLine("Выберите способ доставки."); } Console.WriteLine("Стоимость доставки: " + deliveryCost);

В данном примере мы определяем переменную deliveryMethod со значением "авиа". Затем мы проверяем, какой способ доставки выбран. Если выбран способ доставки "авиа", то мы устанавливаем переменную deliveryCost в значение 1000. Если выбран способ доставки "ж/д", то мы устанавливаем переменную deliveryCost в значение 500. Если выбран способ доставки "автобус", то мы устанавливаем переменную deliveryCost в значение 300. Если ни одно из условий не выполняется, то мы выводим сообщение "Выберите способ доставки." Наконец, мы выводим на экран стоимость доставки.

Операторы сравнения

Операторы сравнения позволяют сравнивать значения и возвращать булево значение (true или false) в зависимости от того, выполняется ли условие или нет. В C# доступны следующие операторы сравнения:

  • == (равенство)
  • != (неравенство)
  • > (больше)
  • < (меньше)
  • >= (больше или равно)
  • <= (меньше или равно)

Давайте рассмотрим примеры использования операторов сравнения. Допустим, мы хотим проверить, является ли переменная age больше или равной 18:

1

int age = 20; if (age >= 18) { Console.WriteLine("Вы совершеннолетний"); } else { Console.WriteLine("Вы несовершеннолетний"); }

В данном примере мы определяем переменную age со значением 20. Затем мы проверяем, является ли age больше или равной 18. Если это так, то мы выводим сообщение "Вы совершеннолетний". В противном случае, мы выводим сообщение "Вы несовершеннолетний".

Также можно использовать операторы сравнения для сравнения строк. Давайте рассмотрим пример, в котором мы проверяем, является ли переменная name равной "John":

1

string name = "John"; if (name == "John") { Console.WriteLine("Привет, John!"); } else { Console.WriteLine("Вы не John!"); }

В данном примере мы определяем переменную name со значением "John". Затем мы проверяем, является ли name равной "John". Если это так, то мы выводим сообщение "Привет, John!". В противном случае, мы выводим сообщение "Вы не John!".

Логические операторы

Логические операторы позволяют комбинировать несколько условий в одном выражении. В C# доступны следующие логические операторы:

  • && (логическое "И")
  • || (логическое "ИЛИ")
  • ! (логическое "НЕ")

Давайте рассмотрим пример использования логических операторов. Допустим, мы хотим проверить, является ли переменная age больше или равной 18 И name равной "John":

1

int age = 20; string name = "John"; if (age >= 18 && name == "John") { Console.WriteLine("Привет, John! Вы совершеннолетний."); } else { Console.WriteLine("Вы не John или не совершеннолетний."); }

В данном примере мы определяем переменную age со значением 20 и переменную name со значением "John". Затем мы проверяем, является ли age больше или равной 18 И name равной "John". Если это так, то мы выводим сообщение "Привет, John! Вы совершеннолетний.". В противном случае, мы выводим сообщение "Вы не John или не совершеннолетний.".
